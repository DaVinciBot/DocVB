## Overleaf

- Avantages: zéro installation, collaboration, historique, compilation serveur
- Limites: dépendance réseau, packages parfois restreints (p. ex. `minted` sans option shell-escape)

## Local

- Installer TeX Live (Linux/macOS) ou MiKTeX (Windows), éditeur VS Code + extension LaTeX Workshop
- Avantages: pleine liberté (moteurs XeLaTeX/LuaLaTeX), performances, intégration Git
- Inconvénients: installation plus lourde

:::tip Dans ce cours
Nous utiliserons Overleaf par défaut. Une section “Local” est présente pour aller plus loin.
:::
