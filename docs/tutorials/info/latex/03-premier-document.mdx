Crée un projet Overleaf “Parcours LaTeX”. Fichier `main.tex` minimal:

```tex
\documentclass[11pt,a4paper]{article}
\usepackage[T1]{fontenc}
\usepackage[utf8]{inputenc}
\usepackage[french]{babel}
\usepackage{microtype}
\usepackage[margin=2.5cm]{geometry}

\title{Mon premier document}
\author{Prénom Nom}
\date{\today}

\begin{document}
\maketitle
\tableofcontents
\section{Introduction}
Texte avec \emph{emphase} et \textbf{gras}. Liste:\\
\begin{itemize}
  \item point 1
  \item point 2
\end{itemize}
\end{document}
```

:::exercise Exercice (10 min)
- Ajoute une section et une sous-section
- Crée une liste numérotée
- Compile et vérifie la table des matières
:::

:::note Accents
Avec Overleaf moderne et `babel`, l’entrée UTF-8 fonctionne nativement.
:::
