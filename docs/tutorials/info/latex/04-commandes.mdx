## Syntaxe

- Commande: `\nomcommande[option]{argument}`
- Environnement: `\begin{env} ... \end{env}`
- Commentaires: `%`, caractères spéciaux: `\%`, `\_`, `\$`, `\&`, `\{`, `\}`

## Environnements utiles

- Listes: `itemize`, `enumerate`, `description`
- Citations: `quote`, `quotation`
- Alignements: `center`, `flushleft`, `flushright`
