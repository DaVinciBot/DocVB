- Beamer (présentations): classe `beamer`, thèmes, blocs
- TikZ/pgfplots: schémas, diagrammes, courbes
- Glossaires/acronymes: `glossaries`
- Index: `makeidx`
- Multi-bibliographies: `biblatex` avec refsections

Chaque sous-module peut faire l'objet d'un atelier dédié de 30-60 min.
